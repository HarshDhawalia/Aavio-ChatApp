# 💬 Aavio ChatApp

A full-stack real-time chat application built with Spring Boot and React.

## Tech Stack
### Backend
Java 17, Spring Boot, MongoDB, JWT, WebSockets (STOMP), Maven

### Frontend
React.js, STOMP.js, SockJS, Axios, React Router

## Features
- 📱 Login with Mobile Number or Email
- 🔒 JWT Authentication
- ⚡ Real-time messaging with WebSockets
- 👥 Private messaging between users
- 🟢 Online/Offline status
- 💬 Group chat rooms
- 📜 Message history

## Setup
### Backend
1. Install MongoDB and Java 17
2. Run: `cd backend && ./mvnw spring-boot:run`

### Frontend
1. `cd frontend && npm install`
2. `npm start`
3. Open `http://localhost:3000`
